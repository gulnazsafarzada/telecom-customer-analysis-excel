# 📞 Telekom Tarif və Müştəri Davranışları Analizi (Excel)

Bu layihədə telekommunikasiya şirkətinin 2022-2023-cü illər üzrə data bazası əsasında tariflərin stabilliyi, müştəri loyallığı və gəlir optimallaşdırılması analiz edilmişdir.

## 🚀 Analiz Metodologiyası:

### 1. Cohort Analizi (Retention Analysis)
* Müştərilərin kampaniyalara ilk qoşulma tarixlərinə əsasən 2022-2023-cü illəri əhatə edən **Retention (Müştəri saxlama)** hesabatı hazırlanıb.
* Kampaniyalar üzrə filtrləmə imkanı yaradılaraq, hər bir tarifin müştəri loyallığına təsiri ölçülüb.

### 2. Gəlir Stabilliyinin Yoxlanılması
* Kampaniyaların illər və aylar üzrə gəlir dinamikası analiz edilib.
* 2023-cü ilin son 5 tam ayı (Yanvar-May) əsas götürülərək **ən stabil gəlir gətirən TOP 3 kampaniya** müəyyən edilib və şərti formatlaşdırma (Conditional Formatting) tətbiq olunub.

### 3. Pareto Analizi (80/20 Qaydası)
* Gəlirin 80%-ni təmin edən kritik kampaniyaları tapmaq üçün **Pareto Analizi** tətbiq edilib. Bu, resursların hansı tariflərə yönəldilməli olduğunu müəyyən etməyə kömək edir.

### 4. Müştəri Endirim Siyasəti (Logic Testing)
* Müştərilərin son qoşulduğu tariflə, onların tarixçəsindəki ən bahalı tarif müqayisə edilib.
* Mürəkkəb Excel funksiyaları vasitəsilə hansı müştərilərə endirim tətbiq olunub-olunmayacağı avtomatlaşdırılmış şəkildə müəyyən edilib.

## 🛠 Texniki Bacarıqlar:
* **Funksiyalar:** XLOOKUP, VLOOKUP, INDEX-MATCH, IF/IFS, EOMONTH.
* **Analitika:** Cohort Table, Pareto Chart, Trend Analysis.
* **Vizualizasiya:** Dinamik Dashboard, Şərti formatlaşdırma.
